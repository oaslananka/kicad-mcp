# PCB Review Checklist

## 1. Preliminary Checks
- [ ] Project opens without errors
- [ ] KiCad version matches project requirements
- [ ] All footprint libraries are available

## 2. Schematic Review
- [ ] ERC passes with no errors
- [ ] All nets are connected
- [ ] No dangling wires
- [ ] Power symbols are correct
- [ ] Component values are specified
- [ ] Pin swaps are documented

## 3. PCB Layout Review
- [ ] DRC passes with no errors
- [ ] Board outline is complete
- [ ] All components are placed
- [ ] Clearance rules are met
- [ ] Track widths meet current requirements
- [ ] Via sizes are appropriate
- [ ] Copper pour is connected

## 4. Manufacturing Checks
- [ ] Minimum track/space meets fab capabilities
- [ ] Minimum drill size meets fab capabilities
- [ ] No soldermask slivers
- [ ] Silkscreen is legible
- [ ] Panelization marks if needed
- [ ] Edge clearance for routing

## 5. Documentation
- [ ] Assembly drawing
- [ ] Fabrication drawing
- [ ] Stackup document
- [ ] BOM is complete with MPNs
